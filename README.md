# Hands-on-training-in-the-Antimicrobial-Potential-of-Plant--Derived-Metabolites-against-Oral-Pathogen
A practical workshop exploring how plant-derived secondary metabolites — such as alkaloids, flavonoids, and terpenes — inhibit oral pathogens like Streptococcus mutans and Candida albicans, through laboratory techniques including MIC assays, disk diffusion, and biofilm inhibition studies.
